## 一、Node.js 是什么？

Node.js 是一个基于 Chrome V8 引擎的 **JavaScript 运行环境**。  
它采用事件驱动和非阻塞式 I/O 模型，适合构建高性能、分布式应用。

通俗来说，Node.js 让 JavaScript 不再局限于浏览器端，而是可以在服务端与 PHP、Python、Ruby 等语言平起平坐。归根到底，它就是一个 JavaScript 脚本语言的运行环境。

## 二、npm 是什么？

`npm` 是 Node.js 默认的包管理工具（package manager），类似于木工程里的 Maven。  
它让我们在开发过程中，能够通过简单命令安装第三方模块，而不用手动下载、解压、管理依赖。

当模块之间存在相互依赖关系，npm 会自动处理依赖树，极大减轻维护难度。
